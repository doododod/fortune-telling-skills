## 各宫详解

### 命宫
- **核心**: 一生格局、性格本质、先天潜力
- **北派分析要点**:
  1. 命宫主星庙旺利陷判定先天格局高低
  2. 三方四正（命、事业、财帛、迁移）会合的星曜总论格局
  3. 命宫四化（生年四化落入命宫的影响）
  4. 命宫是否成特殊格局（如紫府朝垣、日月并明等）
- **南派补充**: 命宫宫干飞四化，看化禄、化忌分别飞入哪宫，判断一生重心与困扰方向

### 兄弟宫
- **核心**: 兄弟关系、合伙人、母亲（兄弟宫为父母宫的夫妻宫）、现金流
- **北派分析要点**:
  1. 主星判断兄弟数量与关系亲疏
  2. 煞星多主兄弟有损或关系不和
  3. 现金流角度：兄弟宫为财帛宫的田宅宫，看流动资金
- **南派补充**: 兄弟宫化忌冲交友宫，主合伙破裂或人际受损

### 夫妻宫
- **核心**: 配偶特征、婚姻质量、感情模式
- **北派分析要点**:
  1. 主星判断配偶个性与条件
  2. 桃花星（贪狼、廉贞、天姚、红鸾等）多则感情复杂
  3. 煞星多主婚姻波折，化忌入或冲夫妻宫主感情障碍
  4. 夫妻宫对宫为事业宫，事业与感情需平衡
- **南派补充**: 夫妻宫飞化忌入命宫，配偶对命主执着（可能控制欲强）；化忌冲命宫，则配偶带来困扰

### 子女宫
- **核心**: 子女缘分、桃花、合伙投资、性生活
- **北派分析要点**:
  1. 主星判断子女数量与关系
  2. 子女宫也是"桃花宫"，看感情际遇
  3. 子女宫为投资的参考（子女宫为财帛宫的交友宫）
- **南派补充**: 子女宫化忌冲田宅宫，主投资损失或家庭不安

### 财帛宫
- **核心**: 赚钱能力、财源方向、理财方式
- **北派分析要点**:
  1. 主星判断财源方向（正财/偏财/技术财等）
  2. 化禄入财帛主财源丰沛，化忌入主赚钱辛苦
  3. 三方（命宫、事业宫）配合看赚钱格局完整性
  4. 禄存在财帛宫主稳定正财收入
- **南派补充**: 财帛宫飞化忌入各宫看花钱方向。化忌入疾厄宫主花钱在健康

### 疾厄宫
- **核心**: 身体健康、体质、先天弱点
- **北派分析要点**:
  1. 主星五行判断体质偏向与易患疾病
  2. 煞星多主体质差或意外伤害
  3. 化忌入疾厄主慢性病或心理困扰
  4. 疾厄宫也看EQ与情绪管理
- **南派补充**: 疾厄宫为"一六共宗"（命宫的气数位），飞化影响整体运势

### 迁移宫
- **核心**: 外出运、社交能力、外在表现、贵人运
- **北派分析要点**:
  1. 迁移宫吉星多主外出有利、社交能力强
  2. 迁移宫煞星多主在外不利、出外小心
  3. 迁移宫与命宫对看，判断内外差异
  4. 流年迁移宫逢吉主出行有利
- **南派补充**: 迁移宫化禄入命主外来财/缘进入命宫，化忌冲命则外在压力

### 交友宫（奴仆宫）
- **核心**: 朋友关系、下属、社交圈质量
- **北派分析要点**:
  1. 主星判断朋友层次与关系
  2. 煞星多主交友不慎或被下属/朋友拖累
  3. 化忌入主被人利用或朋友带来麻烦
  4. 交友宫也看团队管理能力
- **南派补充**: 交友宫为命宫的"成就位"，交友宫四化影响人生成就

### 事业宫（官禄宫）
- **核心**: 事业方向、工作模式、社会成就
- **北派分析要点**:
  1. 主星判断适合的职业方向
  2. 三方（命宫、财帛宫）配合看事业格局
  3. 化权入事业宫主掌权有成就
  4. 事业宫与夫妻宫对看，判断事业与感情平衡
- **南派补充**: 事业宫飞四化为判断事业成败的关键

### 田宅宫
- **核心**: 不动产、居住环境、家庭、存款
- **北派分析要点**:
  1. 主星判断房产运势
  2. 化禄入主买房/置产顺利
  3. 化忌入或空劫入主房产纠纷或存款不足
  4. 田宅宫为"财库"位，也看积蓄能力
- **南派补充**: 田宅宫为命盘的"财库"，飞化忌冲子女宫主投资失利

### 福德宫
- **核心**: 精神世界、兴趣爱好、人生享受、思维方式
- **北派分析要点**:
  1. 主星判断精神层次与享受方式
  2. 煞星多主精神困扰、思虑过重
  3. 化忌入主内心不安、精神苦闷
  4. 福德宫为财帛宫对宫，精神满足与物质追求的平衡
- **南派补充**: 福德宫为来世因果宫，飞化看前世业力（南派较重此说）

### 父母宫（相貌宫）
- **核心**: 父母缘分、上司关系、学业、相貌
- **北派分析要点**:
  1. 主星判断父母条件与关系
  2. 父母宫也看学历与考试运
  3. 煞星多或化忌入主与父母缘薄或上司关系不佳
  4. 父母宫为"文书宫"，看合约、证书相关事宜
- **南派补充**: 父母宫为命宫的"因果位"，飞化追踪人生根源

## 宫位关系

### 对宫关系
对宫互为表里，相互影响：
- 命宫 ↔ 迁移宫：内在 vs 外在
- 夫妻宫 ↔ 事业宫：感情 vs 事业
- 财帛宫 ↔ 福德宫：物质 vs 精神
- 子女宫 ↔ 田宅宫：投资 vs 储蓄
- 兄弟宫 ↔ 交友宫：近人 vs 远人
- 疾厄宫 ↔ 父母宫：体质 vs 遗传

### 邻宫夹持
夹持对宫位影响极大：
- **财荫夹印**: 天梁（或化禄）与禄星夹天相 → 大吉
- **刑忌夹印**: 擎羊（或廉贞化忌）与化忌夹天相 → 大凶
- **空劫夹**: 地空地劫夹某宫 → 该宫虚浮不实

## 三方四正

分析任何宫位必须看三方四正（本宫 + 对宫 + 左右三合宫）全部星曜会合：

```
以命宫为例：
        事业宫
        ╱    ╲
    命宫 ——— 迁移宫（对宫）
        ╲    ╱
        财帛宫
```

**北派核心原则**: 三方四正会合的星曜总和才是该宫位的完整格局。单看本宫不足以论断。

## 宫位飞化（南派核心）

### 飞化基本原理
每个宫位的天干可以产生四化，飞入十二宫各有含义：
- **化禄飞入某宫**: 该宫有"缘"，命主在该宫主题上获益
- **化权飞入某宫**: 该宫被"掌控"，命主在该宫主题上施展权力
- **化科飞入某宫**: 该宫有"名"，命主在该宫主题上得声誉/贵人
- **化忌飞入某宫**: 该宫有"困"，命主在该宫主题上受阻/执着

### 关键飞化组合
- **禄转忌**: A宫化禄入B宫，B宫化忌入C宫 → 因B而在C处受损
- **忌冲**: 化忌入某宫，同时冲对宫 → 对宫受冲击更大
- **自化**: 本宫星曜化出（如命宫主星自化忌）→ 主动流失该宫能量
- **双忌叠加**: 生年忌 + 飞星忌同宫 → 凶意加倍

### 宫位借转（南派进阶）
- **借宫**: 以某宫为太极点，重新定义十二宫看不同层面
- **例**: 以夫妻宫为命宫，看配偶的事业 = 夫妻宫的事业宫（即原盘的疾厄宫）
