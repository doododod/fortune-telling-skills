## 富贵格局

### 顶级格局

**紫府朝垣**
- 条件: 紫微天府分别在三方四正会照命宫
- 效果: 帝相得位，主大富大贵
- 关键: 须无煞星冲破，有辅弼魁钺更佳

**日月并明**
- 条件: 太阳在巳/午，太阴在戌/亥，分居命宫三方四正且各居庙旺
- 效果: 阴阳调和，主贵显
- 关键: 日生人太阳要庙，夜生人太阴要庙

**府相朝垣**
- 条件: 天府天相在三方四正会照
- 效果: 稳重富足
- 关键: 天相不可受刑忌夹

**明珠出海**
- 条件: 日巨同宫在寅宫（太阳庙旺化暗为明）
- 效果: 太阳驱散巨门之暗，主名声远播
- 关键: 太阳必须庙旺，加化禄/化权更佳

**石中隐玉**
- 条件: 巨门在子宫坐命
- 效果: 巨门入庙，暗中藏贵气
- 关键: 不受煞星冲破方成格

### 暴发格局

**火贪格/铃贪格**
- 条件: 贪狼与火星（或铃星）同宫或会照
- 效果: 主暴发、横财、意外之财
- 关键: 在辰戌丑未四库最佳。但暴发后能否守财需看天府/禄存

**武贪横财格**
- 条件: 武曲贪狼同宫
- 效果: 偏财运、投资运
- 关键: 加火铃为暴发，加禄存为实际横财

### 文贵格局

**机月同梁格**
- 条件: 天机、太阴、天同、天梁在命宫三方四正会齐
- 效果: 主公职、专业技术、稳定收入
- 关键: 适合公务员、教师、医疗等稳定职业。忌创业

**阳梁昌禄格**
- 条件: 太阳、天梁在卯宫，加文昌化禄
- 效果: 主学术成就、考试顺利
- 关键: 利文教事业

**文桂文华格**
- 条件: 文昌文曲同宫或会照
- 效果: 才华出众、文采斐然
- 关键: 化科更佳，化忌则文书出错

### 权贵格局

**君臣庆会**
- 条件: 紫微在命宫，左辅右弼天魁天钺文昌文曲会照
- 效果: 百官朝拱，主位高权重
- 关键: 六吉齐聚方成大格

**紫杀同宫（掌权格）**
- 条件: 紫微七杀同宫
- 效果: 有权有将，文武双全
- 关键: 须有辅星扶持，否则帝将矛盾

## 凶煞格局

### 重大凶格

**刑忌夹印**
- 条件: 天相被擎羊（或廉贞化忌）与化忌两邻宫夹持
- 效果: 主刑克、困顿、灾祸
- 判断: 检查天相所在宫的两邻宫是否有擎羊+化忌组合

**马头带箭**
- 条件: 擎羊在午宫坐命
- 效果: 主刑克、波折
- 变格: 若有吉星化解，可成"将军格"（武职有成）

**风流彩杖**
- 条件: 贪狼在命宫，多桃花星会照（天姚、红鸾、咸池等）
- 效果: 感情复杂、桃花纠纷
- 判断: 桃花星越多越凶

**命里逢空**
- 条件: 地空地劫在命宫或三方四正
- 效果: 空想不实、钱财难聚
- 变格: 空劫逢主星庙旺，可成创意/宗教格（空劫好坏需辨证）

### 破败格局

**羊陀夹忌**
- 条件: 化忌被擎羊、陀罗两邻宫夹持
- 效果: 进退失据，主大困
- 判断: 化忌星所在宫的前后宫是否有擎羊陀罗

**火铃夹命**
- 条件: 火星铃星夹命宫
- 效果: 脾气暴烈，多灾多难
- 判断: 命宫两邻宫有火星铃星

**廉贞化忌 + 煞星**
- 条件: 廉贞化忌遇擎羊或火星
- 效果: 血光之灾、官非牢狱（最凶组合之一）
- 判断: 丙年生人廉贞化忌，检查是否有煞星同宫

## 特殊格局

### 宗教/哲学格局
- **条件**: 天机/天梁入命 + 空劫 + 华盖
- **特征**: 适合宗教、哲学、玄学研究
- **判断**: 主星偏精神类 + 空劫化解物质执着

### 艺术格局
- **条件**: 文昌文曲 + 贪狼 + 天姚/咸池等桃花星
- **特征**: 艺术天赋极高，适合演艺、文创
- **判断**: 才艺星与桃花星组合

### 医药格局
- **条件**: 天梁 + 天刑 + 太阴/天机
- **特征**: 适合医疗、药学、心理咨询
- **判断**: 荫星+刑星+细腻星

## 星化合参体系

### 什么是星化合参
星化合参是紫微斗数最核心的分析方法，融合北派（星情论断）与南派（飞星四化）两大体系，通过"星曜特质 + 四化动态 + 宫位关系"三维分析得出完整论断。

### 星化合参分析步骤

#### 第一层：星情基础（北派为主）
1. **定格局**: 命宫主星庙旺利陷 → 先天格局高低
2. **看三方**: 三方四正会合星曜 → 格局完整性
3. **论组合**: 主星+辅星+煞星的搭配 → 具体性格与能力
4. **辨吉凶**: 吉星多寡、煞星制化 → 顺逆程度

#### 第二层：四化动态（南派为主）
1. **生年四化**: 命主与生俱来的禄权科忌分布 → 先天优势与业力
2. **宫干飞化**: 各宫天干飞出四化 → 宫位间的因果链
3. **化忌追踪**: 化忌飞入与冲破 → 困难与障碍的根源
4. **禄忌对应**: 禄在哪里就在哪里有缘，忌在哪里就在哪里受困

#### 第三层：合参综论
1. **星情 × 四化**: 主星特质在四化作用下的具体表现
   - 例：武曲化忌 = 财星受困 → 具体表现为破财、周转不灵
   - 例：太阳化禄入命 = 光明星得禄 → 贵人多、社交广
2. **格局 × 飞化**: 格局被飞化增强或破坏
   - 例：火贪格 + 命宫飞化禄入财帛宫 → 格局加倍，暴发力更强
   - 例：府相朝垣 + 化忌冲天相 → 格局被破，表面好实则困
3. **宫位 × 星化**: 具体宫位中星曜与四化的合参
   - 例：夫妻宫有贪狼化忌 → 婚姻有桃花劫（星情）+ 配偶带来困扰（飞化）

### 合参分析示例模板

```
【命宫分析】
星情: [主星] 在 [宫位]，[庙旺利陷]，[组合星曜]
四化: 生年 [化X] 入命宫，命宫干飞 [化X] 入 [某宫]
合参: [综合论断]

【事业分析】
星情: 事业宫 [主星]，三方会 [星曜列表]
四化: 事业宫干飞化 [化X] 入 [某宫]，生年 [化X] 入事业宫
合参: [综合论断]
```

## 大运流年分析

### 大运（十年运）
- **排法**: 以命宫为起点，顺行或逆行（阳男阴女顺行，阴男阳女逆行）
- **分析**: 大运宫位的星曜 + 大运天干飞四化 + 与本命盘的交叉影响
- **北派**: 重大运宫内星曜的庙旺利陷与吉凶星组合
- **南派**: 重大运四化飞入本命各宫的因果关系

### 流年分析
- **排法**: 以流年地支对应的宫位为流年命宫
- **分析层次**:
  1. 流年命宫星曜（本命盘该宫星曜）
  2. 流年四化（流年天干产生的四化）
  3. 大运四化叠加
  4. 本命四化叠加
- **三叠法**: 本命 + 大运 + 流年三重四化叠加分析

### 流年事件判断
- **化禄入某宫**: 该年该宫主题获益/有缘
- **化忌入某宫**: 该年该宫主题受困/出问题
- **化忌冲某宫**: 比入宫更凶，该宫受冲击
- **双忌叠加**: 本命忌 + 流年忌同宫 = 该年该宫主题大凶
- **禄忌同宫**: 有好有坏，得失参半

### 关键流年事件指标
| 事件 | 指标 |
|------|------|
| 结婚 | 流年化禄/化科入夫妻宫，红鸾天喜到命宫/夫妻宫 |
| 升职 | 流年化权入事业宫/命宫，化禄入命宫 |
| 破财 | 流年化忌入/冲财帛宫或田宅宫 |
| 健康 | 流年化忌入/冲疾厄宫，煞星叠加 |
| 搬家 | 流年化忌冲田宅宫或天马入田宅宫 |
| 感情变动 | 流年化忌入/冲夫妻宫，桃花星到夫妻宫 |
