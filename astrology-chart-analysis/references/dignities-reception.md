## 必然尊贵（Essential Dignities）

行星因所在星座获得的固有品质。

### 五级尊贵度

| 等级 | 名称 | 力量 | 说明 |
|------|------|------|------|
| +5 | 入庙（Domicile） | 最强 | 行星在自己主宰的星座，如火星在白羊 |
| +4 | 擢升（Exaltation） | 很强 | 行星在被提升的星座，如太阳在白羊19° |
| +3 | 三分主（Triplicity） | 中等 | 按元素和教派分配的主宰权 |
| +2 | 界主（Term/Bound） | 较弱 | 每座分5段，各有界主 |
| +1 | 面主（Face/Decan） | 最弱 | 每座分3段（10°一段），各有面主 |

### 必然无力

| 等级 | 名称 | 力量 | 说明 |
|------|------|------|------|
| -5 | 落陷（Detriment） | 最弱 | 入庙的对面星座，如火星在天秤 |
| -4 | 弱势（Fall） | 很弱 | 擢升的对面星座，如太阳在天秤 |
| 0 | 游走（Peregrine） | 无力 | 无任何尊贵度 — 行星无所依靠 |

### 三分主（Triplicity Rulers）

| 元素 | 日间主 | 夜间主 | 合作主 |
|------|--------|--------|--------|
| 火（白羊、狮子、射手） | 太阳 | 木星 | 土星 |
| 土（金牛、处女、摩羯） | 金星 | 月亮 | 火星 |
| 风（双子、天秤、水瓶） | 土星 | 水星 | 木星 |
| 水（巨蟹、天蝎、双鱼） | 金星 | 火星 | 月亮 |

（以Dorotheus体系为准）

### 尊贵度的比喻

- **入庙**：主人在自己家 — 完全掌控，资源充足
- **擢升**：贵宾在豪华酒店 — 受到尊敬，但不是家
- **三分主**：朋友家做客 — 舒适但有限度
- **界主**：在熟人处借住 — 基本安全
- **面主**：在陌生地方但穿着体面 — 仅存面子
- **游走**：流浪者 — 无所依靠
- **落陷**：在敌人的地盘 — 处处受限
- **弱势**：被关在低洼处 — 无法发挥

## 偶然尊贵（Accidental Dignities）

行星因宫位、运动状态等获得的外在力量。

### 宫位力量

| 宫位类型 | 力量 | 宫位 |
|---------|------|------|
| 角宫 | 最强 | 1、10、7、4 |
| 续宫 | 中等 | 11、5、2、8 |
| 果宫 | 最弱 | 9、3、6、12 |

**注意**：10宫 > 1宫 > 7宫 > 4宫（角宫内部排序）

### 其他偶然尊贵/无力

**增加力量的因素：**
- 顺行（Direct motion）
- 速度快于平均
- 东方升起（晨星，水金在太阳之前升起）
- 不被太阳灼烧
- 在自己喜乐宫
- 与木星/金星成吉相位
- 与北交合
- Cazimi（太阳中心17分以内）

**减少力量的因素：**
- 逆行（Retrograde）
- 速度慢于平均
- 燃烧（太阳8°以内）
- 在日光下（太阳17°以内）
- 与火星/土星成凶相位
- 与南交合
- 在6、8、12宫（衰败宫）
- 被围困（夹在两凶星之间）

## 互容与接纳（Reception）

### 定义

当两颗行星各自落在对方主宰的星座中时，形成互容。

### 互容类型

#### 1. 庙宫互容（Mutual Reception by Domicile）

最强的互容。例：
- 火星在金牛（金星主宰），金星在白羊（火星主宰）
- 双方互相"接待"对方 → 尽管各自落陷，但有互助基础

#### 2. 擢升互容

例：太阳在天秤（土星擢升），土星在白羊（太阳擢升）

#### 3. 混合互容

一方以庙接纳，另一方以擢升接纳。例：
- 月亮在摩羯（土星庙），土星在巨蟹（月亮庙）→ 这也是庙宫互容
- 月亮在金牛（月亮擢升），金星在巨蟹（木星擢升）→ 不构成互容

#### 4. 单向接纳（Unilateral Reception）

仅一方接纳另一方：
- 火星在金牛（金星主宰），金星在双子（水星主宰）
- 金星接纳了火星，但火星未接纳金星
- 意涵：金星对火星有善意/控制权，但火星不回报

### 接纳在相位中的作用

- **有互容的凶相位**：紧张被缓解，双方有解决问题的意愿
- **无互容的吉相位**：和谐但缺乏深层连接
- **有互容的吉相位**：最佳配置，互相支持且有实际能力帮助
- **无互容的凶相位**：最具挑战，冲突且无缓解机制

## 行星状态综合评估

### 评分框架

对每颗行星进行综合评估：

```
1. 必然尊贵度得分：入庙(+5) / 擢升(+4) / 三分主(+3) / 界主(+2) / 面主(+1)
                   落陷(-5) / 弱势(-4) / 游走(0)
2. 偶然尊贵度：宫位力量、运动状态、与太阳关系
3. 教派状态：是否符合教派
4. 相位接收：受吉星/凶星相位影响
5. 互容关系：是否与其他行星互容
```

### 行星状态的实际意涵

| 状态 | 意涵 |
|------|------|
| 高尊贵+角宫 | 该领域资源充足且行动力强 |
| 高尊贵+果宫 | 有能力但难以外显 |
| 低尊贵+角宫 | 高曝光但品质差 |
| 低尊贵+果宫 | 双重困难，该领域长期挑战 |
| 游走+无互容 | 最不利 — 无资源、无援助 |
| 落陷+有互容 | 困难中有出路 |

### 综合判读示例

```
火星在天秤（落陷 -5）、10宫（角宫 +强）、逆行（-）、与木星三分（+）
→ 事业领域行动力高但方式不佳，有贵人相助缓解
→ 逆行暗示行动方向反复
→ 落陷+角宫 = 高调但问题多，需要木星三分的缓冲
```
