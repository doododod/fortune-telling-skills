## 合盘（Synastry）

将两个人的本命盘叠加，观察行星间的互动。

### 核心要素

#### 1. 轴点互动（最重要）

- A的行星合B的ASC：A对B的自我认知有直接影响
- A的行星合B的MC：A影响B的事业和公众形象
- A的行星合B的DSC：A是B理想伴侣的投射
- A的行星合B的IC：A触动B的内在安全感

#### 2. 发光体互动

| 相位 | 意涵 |
|------|------|
| A日合B月 | 经典的阴阳互补，强烈吸引 |
| A日合B日 | 意志共鸣或竞争 |
| A月合B月 | 情感需求相似，日常和谐 |
| A日冲B月 | 张力中的互补，需要磨合 |
| A日刑B月 | 基本需求冲突，挑战大 |

#### 3. 金星-火星互动

- 金星合火星：强烈的性吸引
- 金星拱火星：自然的情欲流动
- 金星刑火星：激情与冲突并存

#### 4. 土星互动

- 土星合个人行星：承诺与责任，但可能带来压力
- 土星拱个人行星：稳定持久的支持
- 土星刑/冲个人行星：限制与控制的张力

### 宫位覆盖（House Overlay）

A的行星落入B的哪个宫位 → A在B生活的哪个领域产生影响

| A的行星落入B的宫位 | 影响领域 |
|-------------------|---------|
| 1宫 | 直接影响B的自我认知和形象 |
| 4宫 | 影响B的家庭和安全感 |
| 5宫 | 浪漫、创造力、快乐 |
| 7宫 | 伴侣关系直接关联 |
| 8宫 | 深层转化、亲密与共享资源 |
| 10宫 | 影响B的事业和社会地位 |

## 组合盘（Composite Chart）

### 原理

取两人各对应行星的黄道中点，组成一张新盘，代表"关系本身"的能量。

### 关键解读要素

- **组合ASC**：关系的外在面貌和第一印象
- **组合太阳**：关系的核心目的和生命力
- **组合月亮**：关系中的情感基调和安全感
- **组合金星**：关系中爱与价值的表达
- **组合火星**：关系中的行动力和冲突模式
- **组合土星**：关系中的结构、承诺和挑战

### 与合盘的区别

| 方面 | 合盘（Synastry） | 组合盘（Composite） |
|------|-------------------|---------------------|
| 视角 | 两人如何互相影响 | 关系本身的特质 |
| 方法 | 两盘叠加对比 | 中点合成新盘 |
| 适用 | 理解互动动态 | 理解关系主题 |
| 比喻 | 两个人在跳舞 | 舞蹈本身的风格 |

## 古典合盘技法

### 1. 七宫主互动

- A的七宫主与B的行星关系 → A的婚姻期望是否被B满足
- 双方七宫主之间是否有相位 → 关系的基础契合度

### 2. 金星与月亮

古典占星中，金星和月亮是关系的核心显示器：
- 男性盘看金星和月亮 → 显示其理想伴侣类型
- 女性盘看太阳和火星 → 显示其被吸引的伴侣类型
- 交叉对比这些行星的状态和相位

### 3. 接纳与互容（Reception）

- **互容（Mutual Reception）**：A主宰B所在星座，B主宰A所在星座 → 双方互相支持
- **单向接纳**：一方接纳另一方 → 权力不对等
- 在合盘中，互容关系暗示双方有天然的理解和支持基础

### 4. Lot of Marriage

- 男性盘：ASC + Venus - Saturn
- 女性盘：ASC + Saturn - Venus
- 其主宰星和所在宫位显示婚姻的特质

## 分析框架

### 合盘分析步骤

1. **各自本命盘评估**：先单独分析每个人的关系模式（7宫主、金星、月亮状态）
2. **轴点互动**：检查行星与对方轴点的合相
3. **发光体和内行星互动**：日月金火之间的相位
4. **土星互动**：承诺与压力的来源
5. **宫位覆盖**：行星落入对方宫位的分布
6. **整体评估**：吉相位与凶相位的比例，核心张力在哪里

### 输出结构建议

```
## 关系概览（1-2句总结）

## 吸引力来源
- 核心吸引相位
- 宫位覆盖亮点

## 和谐与支持
- 吉相位列表及意涵

## 挑战与张力
- 凶相位及其处理建议

## 长期潜力
- 土星互动分析
- 整体评估
```
