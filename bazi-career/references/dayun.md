## 大运基础

### 大运排法要点
- 大运以月柱为基础排列
- 阳年生男/阴年生女：顺排
- 阴年生男/阳年生女：逆排
- 每步大运管十年（前五年重天干，后五年重地支）

### 大运分析层次
1. **大运天干**: 外在环境、表面趋势
2. **大运地支**: 内在本质、实际力量
3. **大运与原局关系**: 生克冲合刑害
4. **大运对格局的影响**: 助格成格还是破格败格

### 大运对格局的影响判断

**助格运**（好运）:
- 大运走用神：格局得到强化
- 大运走相神：辅助力量增强
- 大运制忌神：阻碍被消除

**破格运**（坏运）:
- 大运走忌神：格局受到破坏
- 大运冲克用神：核心优势被削弱
- 大运合化用神：用神力量被改变

## 十神大运与事业变化

### 比肩运

**事业环境**:
- 同行竞争加剧
- 合伙机会出现
- 独立倾向增强

**吉象**: 比肩为用时 → 竞争中胜出，独立发展有成
**凶象**: 比肩为忌时 → 竞争失利，合伙纠纷，利润被分薄

### 劫财运

**事业环境**:
- 冒险投机机会多
- 社交活跃，人脉扩展
- 合伙动荡

**吉象**: 劫财为用时 → 大胆出击有收获，人脉带来机会
**凶象**: 劫财为忌时 → 投资损失，被人设计，合伙破裂

### 食神运

**事业环境**:
- 技术/才华得到施展
- 工作状态轻松
- 产出增加

**吉象**: 食神为用时 → 专业获认可，作品有成，技术突破
**凶象**: 食神为忌时 → 过于安逸，竞争力下降，被后辈超越

### 伤官运

**事业环境**:
- 创新冲动强烈
- 职场人际关系紧张
- 可能跳槽或转型

**吉象**: 伤官为用时 → 创新突破，独辟蹊径，才华展露
**凶象**: 伤官为忌时（特别是伤官见官）→ 得罪上级，降职，官非

### 正财运

**事业环境**:
- 收入逐步增加
- 工作量加大但有回报
- 务实经营期

**吉象**: 正财为用时 → 薪资增长，实业有成，财务稳健
**凶象**: 正财为忌时 → 为钱辛苦，被财务问题困扰

### 偏财运

**事业环境**:
- 投资机会涌现
- 商业格局扩大
- 人脉带来财缘

**吉象**: 偏财为用时 → 投资获利，商业成功，意外之财
**凶象**: 偏财为忌时 → 投资亏损，上当受骗，财来财去

### 正官运

**事业环境**:
- 升迁机会出现
- 管理责任增加
- 受体制约束加强

**吉象**: 正官为用时 → 升职加薪，社会地位提升，得上级赏识
**凶象**: 正官为忌时 → 受打压，工作束缚大，官非口舌

### 七杀运

**事业环境**:
- 竞争压力骤增
- 事业面临挑战
- 有大破大立的机会

**吉象**: 七杀为用时（有制化）→ 危机中崛起，大权在握
**凶象**: 七杀为忌时（无制化）→ 被打压，失业，严重挫折

### 正印运

**事业环境**:
- 学习进修机会
- 贵人出现
- 工作环境改善

**吉象**: 正印为用时 → 贵人提拔，考试晋升，学术有成
**凶象**: 正印为忌时 → 过于依赖，行动力下降，空有虚名

### 偏印运

**事业环境**:
- 兴趣方向转变
- 接触冷门领域
- 思维方式改变

**吉象**: 偏印为用时 → 冷门领域突破，独特技能获认可
**凶象**: 偏印为忌时（枭神夺食）→ 才华被压制，事业反复

## 流年事业事件判断

### 流年十神引动

**流年官星到位**:
- 升迁、任命、获得职务
- 与原局/大运官星配合看成败

**流年财星到位**:
- 收入变化（增减视用忌）
- 商业机会或财务问题

**流年印星到位**:
- 考试、证照、贵人
- 学习进修、岗位调动

**流年食伤到位**:
- 作品发表、技术突破
- 创业行动、项目启动

**流年比劫到位**:
- 合伙变动、竞争事件
- 同事关系变化

### 流年冲合的事业影响

**流年冲原局**:
- 冲用神：事业受阻，原有优势被打破
- 冲忌神：阻碍被清除，反而有利
- 冲月柱：事业方向大变
- 冲日柱：个人状态大变

**流年合原局**:
- 合用神：用神被合住，力量减弱（不一定好）
- 合忌神：忌神被合住，威胁减轻（通常好）
- 合化改变五行：事业方向可能转变

## 大运流年叠加分析

### 三层叠加法

```
第一层: 原局格局 → 先天事业基调
第二层: 大运十神 → 十年事业大环境
第三层: 流年十神 → 具体年份事件触发
```

### 叠加判断原则

**三层同向吉**（最佳）:
- 原局格局成格 + 大运走用神 + 流年助力
- 该年事业大成

**三层同向凶**（最差）:
- 原局格局有瑕疵 + 大运走忌神 + 流年冲破
- 该年事业大败

**两吉一凶**:
- 凶在原局：有阻碍但可突破
- 凶在大运：大环境不佳但有小成
- 凶在流年：短暂受挫后恢复

**两凶一吉**:
- 吉在原局：有底蕴但困难重重
- 吉在大运：大环境保护但细节不利
- 吉在流年：困境中有短暂喘息

## 关键事业事件指标

### 事件判断速查表

| 事件 | 命理指标 |
|------|----------|
| 升职 | 流年官星得力+大运走官印运+原局官星为用 |
| 降职 | 流年伤官克官+大运走忌神+原局官星受损 |
| 跳槽 | 流年冲月柱或驿马星动+食伤旺 |
| 创业 | 流年食伤生财+大运走食伤或偏财+比劫弱 |
| 破产 | 流年忌神冲财库+大运走比劫争财 |
| 获奖 | 流年印星+食伤得力+大运走用神 |
| 裁员 | 流年七杀无制攻身+大运走忌神 |
| 合伙 | 流年比劫+偏财出现+天合地合 |
| 分伙 | 流年比劫+冲合+忌神得力 |
| 退休 | 大运转印运+流年食伤泄气 |
| 行业转型 | 大运五行变化+流年冲原局格局 |

### 事业高峰判断
1. 大运走用神且力量足够
2. 流年引动原局最佳组合
3. 格局被大运强化至最高状态
4. 财官印形成良好流通

### 事业低谷判断
1. 大运走忌神且力量强大
2. 流年冲破原局格局核心
3. 用神被合化或克伤
4. 忌神肆虐无制约

### 事业转折判断
1. 大运交接之年（换运年）
2. 流年天克地冲
3. 流年与月柱天合地合或天克地冲
4. 三合/三会局改变原局力量格局
