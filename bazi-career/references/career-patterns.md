## 格局与创业/打工判断

### 适合创业的命理特征

**强烈创业信号**:
- 食伤生财格局：才华+赚钱能力，天生创业胚子
- 偏财旺且身强：大格局大手笔，商业天赋
- 七杀有制且身旺：开拓型领袖，不惧竞争
- 比劫旺+食伤泄：竞争意识+产出能力
- 伤官生财：创新变现，适合做颠覆性产品

**中等创业信号**:
- 正财格身旺：可以稳扎稳打做实业
- 偏印格配偏财：冷门领域创业
- 从儿格：以纯粹技术/才华创业

**不利创业信号**:
- 正官格纯粹：更适合体制内发展
- 正印格无食伤：学者型，不善商业
- 比劫争财严重：合伙必出纠纷

### 适合打工（体制内）的命理特征

**强烈体制信号**:
- 官印相生：天生仕途命
- 机月同梁格（借用紫微概念）= 正官配正印食神：公职稳定
- 正官格成格：管理岗天赋
- 正财格：薪资导向型，适合大企业

**中等体制信号**:
- 七杀配印：可以在体制内做技术型领导
- 食神制杀：以技术获得管理岗位

### 创业vs打工的决策因子

| 因素 | 倾向创业 | 倾向打工 |
|------|----------|----------|
| 格局 | 食伤生财、偏财格 | 官印相生、正官格 |
| 日主强弱 | 身旺（能担财官） | 身弱（需外部支持） |
| 比劫 | 弱（少合伙纠纷） | 中等（有同事协作） |
| 食伤 | 旺（有产品输出） | 弱（不需独立创造） |
| 官杀 | 弱或有制（不受束缚） | 旺（受管理约束有成） |
| 大运 | 走食伤财运时创业 | 走官印运时入职 |

## 合伙运分析

### 看合伙的宫位与十神

**比肩**: 同性别合伙人，平等合作
- 比肩为用：合伙有利
- 比肩为忌：竞争大于合作
- 比肩争财：分利严重

**劫财**: 异性别合伙人或冒险型伙伴
- 劫财为用：合伙可带来机会
- 劫财为忌：合伙必出纠纷，被合伙人坑

**月柱比劫**: 青壮年合伙运
**日支比劫**: 配偶参与合伙
**时柱比劫**: 晚年合伙或与晚辈合伙

### 合伙吉凶判断

**适合合伙**:
- 比肩为用神，且有官杀制约
- 偏财旺+比肩适度：人脉合作型
- 食伤+比肩：技术合伙团队

**不适合合伙**:
- 比劫争财严重
- 劫财为忌且无制
- 四柱比劫过重

## 升迁时机分析

### 升迁的命理信号

**大运层面**:
- 走官印运：权力+贵人，升迁概率最高
- 走财运（财生官）：因业绩获升迁
- 走食伤运（食伤制杀）：因技术获管理岗

**流年层面**:
- 流年官星透出或得力：该年有升迁机会
- 流年印星生身：得贵人提拔
- 流年财星生官：因业绩得赏识

### 升迁受阻的信号

- 流年伤官克官：与上级冲突，升迁受阻
- 流年比劫争财：竞争者抢先
- 流年忌神得力：该年不利仕途
- 大运走忌神运：整个阶段事业不顺

### 升迁时机判断模板

```
1. 原局是否有官星（有升迁基础）
2. 原局官星是否为用（升迁是否对命主有利）
3. 大运是否走官印运（时势是否配合）
4. 流年是否引动官星（具体年份触发）
5. 是否有伤官克官的流年干扰
```

## 事业转型判断

### 转型时机信号

**大运换运**:
- 大运五行变化 = 事业方向可能变化
- 如：从官印运转入食伤运 → 从管理转向技术/创业
- 从食伤运转入财运 → 从研发转向商业变现
- 从财运转入官运 → 从经商转入管理/体制

**流年冲破**:
- 流年冲命局用神 → 被迫转型
- 流年合化改变格局 → 主动转型机遇

### 转型方向判断

1. **新大运十神** → 新事业方向
2. **新大运五行** → 新行业领域
3. **原局喜用** → 始终适合的底层方向
4. **食伤星** → 新技能/新产品的方向

### 转型吉凶

- 顺势转型（大运方向与原局喜用一致）→ 成功概率高
- 逆势转型（大运方向与原局喜用冲突）→ 风险大
- 被迫转型（忌神冲破用神）→ 先苦后甜or持续困难

## 财运与事业联动

### 正财与事业

**正财为用**:
- 事业以稳定收入为导向
- 适合薪资型工作或稳扎稳打的生意
- 财来自日常经营和劳动

**正财为忌**:
- 赚钱辛苦，付出多回报少
- 或被财务问题困扰

### 偏财与事业

**偏财为用**:
- 事业以投资、人脉、大笔交易为导向
- 适合投资型或人脉型事业
- 财来自机会把握和人际关系

**偏财为忌**:
- 投资风险大，容易被骗
- 人脉带来的是麻烦而非财富

### 财库分析（辰戌丑未）

**四库与财富积累**:
- 命中有财库（日主所克五行之库）：有积蓄能力
- 财库逢冲开：某大运/流年财库被冲开，大量财富进出
- 财库逢合闭：财富被锁住，难以变现

**财库在四柱的位置**:
- 年柱财库：祖上有财产
- 月柱财库：事业核心期积财能力强
- 日柱财库：个人理财能力好
- 时柱财库：晚年积蓄丰厚

### 财星与格局的关系

| 格局 | 财星作用 | 事业财运联动 |
|------|----------|------------|
| 正官格 | 财生官 | 因业绩获升迁+加薪 |
| 七杀格 | 财生杀（忌） | 财多反而事业压力大 |
| 食神格 | 食伤生财 | 才华→产品→利润 |
| 伤官格 | 伤官生财 | 创新→商业价值 |
| 正印格 | 财克印（忌） | 追求财富损害学术 |
| 偏印格 | 财制枭（喜） | 经济压力反而激发潜力 |
| 比劫旺 | 需官制比 | 需要管理约束才能守财 |
